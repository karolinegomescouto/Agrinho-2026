# Agrinho-2026
O presente trabalho consiste no desenvolvimento de um site informativo denominado **"Agro Forte, Futuro Sustentável"**, criado com o objetivo de conscientizar os visitantes sobre a importância da sustentabilidade, da preservação ambiental e do conhecimento para o desenvolvimento responsável do campo. O projeto busca demonstrar como pequenas ações sustentáveis e a adoção de práticas agrícolas conscientes podem contribuir para a proteção dos recursos naturais e para a construção de um futuro melhor.

O site foi desenvolvido utilizando as tecnologias **HTML**, **CSS** e **JavaScript**. O HTML foi empregado para estruturar o conteúdo da página, organizando títulos, textos, seções e informações do quiz. O CSS foi utilizado para criar o design visual, aplicando cores, imagens de fundo, efeitos visuais e responsividade para diferentes dispositivos. Já o JavaScript foi responsável pela interatividade do site, permitindo que o botão "Saiba Mais" exibisse ou ocultasse informações complementares de forma dinâmica e agradável ao usuário.

Para a construção e testes do projeto, foi utilizado um ambiente de desenvolvimento para páginas web, onde os arquivos **index.html**, **style.css** e **script.js** foram organizados e integrados. O resultado é um site moderno, responsivo e educativo, que valoriza o agronegócio sustentável e destaca a importância da preservação ambiental para as futuras gerações.

O site funciona de forma simples e interativa. Ao acessar a página, o usuário encontra informações sobre sustentabilidade e agricultura responsável. O botão **"Saiba Mais"** permite visualizar um texto complementar sobre o tema, podendo ser ocultado novamente ao clicar em **"Mostrar Menos"**. Além disso, o site apresenta dicas de sustentabilidade, benefícios da agricultura sustentável e um quiz educativo com perguntas e respostas, tornando o aprendizado mais dinâmico e acessível.
